# Tempo Plugin Documentation

This repository hosts the compiled static assets for the Tempo Plugin documentation website.

> [!WARNING]
> This repository is the target of an automated CI/CD deployment pipeline. 
> The `main` branch is force-updated automatically when documentation changes in the main repository.
> 
> **Please do not open Pull Requests, create Issues, or push code directly to this repository.**

To contribute to the documentation or report issues, please visit the public [`magma`](https://github.com/magmacomputing/magma) repository.
